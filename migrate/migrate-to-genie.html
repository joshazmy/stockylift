How to move your Stocky data to Genie

Genie is subscription (recurring monthly plans): An inventory and purchase-order management app for Shopify, sold as an ongoing subscription rather than a one-time export tool.

  1. Export your Stocky suppliers with StockyLift. Paste your Stocky API key into the extension, run the export, and download a CSV (or JSON backup) of your full supplier list — the export stocky suppliers button Stocky itself never had. Get StockyLift →
  2. Check what the CSV contains. The Generic CSV preset includes all 18 supplier fields Stocky's API exposes (name, contact, email, phone, address, currency, payment terms, tax number, notes, status, lead time, and more) plus your purchase-order export.
  3. Import into Genie. Sign in to Genie, locate its supplier or purchase-order import tool (usually under Settings → Import, or a Suppliers/Purchasing section), and upload the exported file. Consult Genie's own help center for its current import template — field names vary between apps, so map columns before a large import. Find Genie on the Shopify App Store →
  4. Reconcile the count. Confirm the number of suppliers imported into Genie matches the number exported from Stocky, then keep your original CSV/JSON files as a permanent backup.

Do this before August 31, 2026 — that is when Stocky's API goes offline for good (the stocky api sunset). See the full migration checklist for every deadline.

StockyLift is an unofficial tool and is not affiliated with Shopify.
